Automação de Testes para API REST

Siga todos os passos das pré-condições para que você possa desenvolver os testes durante o workshop. Em caso de dúvidas, entre em contato comigo.

Lista de pré-condições
  • Instalar Node.js
    1. Acessar o site https://nodejs.org
    2. Clique no menu Downloads
    3. Clique sobre o nome da plataforma desejada para instalação
    4. Aguarde o download do arquivo instalador
    5. Clique no arquivo executável e siga os passos propostos pelo instalador
    6. Abra o Prompt de Comando ou Terminal e digite node —help e pressione ENTER
    7. A primeira linha retornada deve ter o texto “Usage: node [options] [ -e script | script.js ] [arguments]”

  • Baixar o projeto base
    1. Acesse a página do projeto https://github.com/eliasnogueira/workshop-api-rest-frisby-base
    2. Se você está familiarizado com Git, faça com clone do projeto através do comando git clone https://github.com/eliasnogueira/workshop-api-rest-frisby-base.git em um local de fácil acesso
    3. Se você não está familiarizado com Git clique no botão Clone or download e depois no link Download. Será necessário descompactar o projeto em algum local de fácil acesso

  • Instalar dependências de execução do projeto
    1. Abra o Prompt de Comando ou terminal e digite npm install -g jasmine-node
    2. Aguarde até ser exibido uma lista de todas as dependências instaladas

  • Instalar as dependências do projeto
    1. Abra o Prompt de Comando e navegue até a pasta do projeto (comando cd)
    2. Execute o seguinte comando npm install e pressione ENTER
    3. O Prompt de Comando ou Terminal vai apresentar o texto “restify escutando em http://[::]:3000” No sistema operacional Windows você receberá uma mensagem do boqueio de acesso. Na tela com esta informação clique no botão Permitir acesso

  • Instalar o Postman
    1. Acesse a página https://www.getpostman.com/
    2. Clique em um dos botões referente a plataforma que você deseja baixar
    3. Aguarde o download do arquivo instalador
    4. Clique no arquivo executável e siga os passos propostos pelo instalador
    5. Após a instalação o Postman será aberto
    6. Faça o cadastro (gratuito) ou efetue o login se você já possui uma conta no Postman
